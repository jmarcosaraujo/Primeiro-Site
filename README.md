# Loja de Peças Gamer

Projeto simples de site de vendas de componentes para computadores gamers.

## Descrição

- Página construída em HTML e CSS puro.
- Conteúdo com seções: processadores, placas de vídeo, memória RAM, armazenamento e refrigeração.
- Estrutura pensada para evolução em JavaScript e backend.

## Como usar

1. Abra `index.html` no navegador.
2. Faça alterações nas seções ou adicione novos produtos.

## Estrutura de arquivos

- `index.html` - conteúdo do site
- `style.css` - estilos visuais
